Sentry Web API 用于以编程方式访问 Sentry 平台。您可以使用 API 来管理帐户级资源，如组织和团队，以及管理和导出数据。

如果您正在寻找有关 Sentry SDK 的 API 表面的信息，请参阅 [SDK 开发](https://develop.sentry.dev/sdk/overview/) 文档。

## 版本控制

当前版本的 Sentry Web API 被认为是 **v0**。我们的公共端点通常是稳定的，但 beta 端点可能会发生变化。

## 选择正确的 API 基础域

虽然我们的许多 API 使用 `sentry.io` 作为 API 端点的主机，但如果您想指定特定的[数据存储位置](/organization/data-storage-location/#what-types-of-data-are-stored-where)，您应该使用特定区域的域。

- 美国区域托管在 `us.sentry.io`
- 德国区域托管在 `de.sentry.io`。

要了解哪些 API 资源在基于区域的域上可用，请参阅[哪些类型的数据存储在哪里](/organization/data-storage-location/#what-types-of-data-are-stored-where)以获取更多信息。
