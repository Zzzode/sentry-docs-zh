## 与 Expo 仪表板的 Sentry 集成

<Alert level="warning">

如果您更改了组织标识符，您需要更新此集成的配置。在我们的[故障排除指南](/organization/integrations/troubleshooting)中了解更多信息。

</Alert>

与 Expo 的 Sentry 集成允许您直接在 Expo 应用服务 (EAS) 仪表板中查看 Expo 应用部署的崩溃报告和[会话重放](/platforms/react-native/session-replay/)。此集成提供了指向 Sentry 堆栈跟踪的完整上下文、会话重放和调试功能的直接链接。

### 安装

<Alert>

需要 Sentry 所有者、管理员或管理员权限才能安装此集成。

</Alert>

1. 登录到您的 Expo 仪表板并打开**账户设置 > 概览** (`https://expo.dev/accounts/[您的账户]/settings`)。
2. 找到**连接**部分，然后点击 Sentry 旁边的**连接**。
3. 登录到 Sentry 并接受集成到您的组织中。您将被重定向回账户设置 > 概览。

### 配置

#### 链接您的项目

连接您的账户后，您需要将您的 Expo 仪表板项目链接到您的 Sentry 项目：

1. 在 Expo 仪表板中打开**项目 > [您的项目] > 配置 > 项目设置**。
2. 点击**链接**并从下拉菜单中选择您的 Sentry 项目。

### 使用

要在 Expo 仪表板中查看您的 Sentry 问题和重放，您首先需要确保已创建[Expo 仪表板频道](https://docs.expo.dev/eas-update/how-it-works/)并将构建分配给它以创建[EAS 部署](https://docs.expo.dev/eas-update/deployment/)。完成后，您可以通过以下方式查看您的 Sentry 数据：

1. 打开**项目 > [您的项目] > 更新 > 部署 > [部署]**以从发布中查看 Sentry 数据。


<Alert level="info">
  要利用此集成，您需要将 Sentry React Native SDK 添加到您的 Expo 应用程序中。您可以通过运行 Sentry 向导轻松设置 SDK：

  ```bash
  npx @sentry/wizard@latest -i reactNative
  ```

  有关在 Expo 中使用 Sentry 的更多详细信息，请参阅 React Native 应用程序的 [Sentry 文档](/platforms/react-native/)。
</Alert>
