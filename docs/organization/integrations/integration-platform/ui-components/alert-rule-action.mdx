警报规则操作组件允许您在创建或编辑警报规则时添加自定义操作。这些操作将出现在警报规则的"操作"部分。

## 架构

```json {filename:schema.json}
{
  "type": "alert-rule-action",
  "title": "Send to My Service",
  "settings": {
    "type": "alert-rule-settings",
    "uri": "/sentry/alert-rule",
    "required_fields": [
      {
        "type": "select",
        "name": "team",
        "label": "Team",
        "uri": "/sentry/teams"
      }
    ],
    "optional_fields": [
      {
        "type": "text",
        "name": "channel",
        "label": "Channel"
      }
    ]
  }
}
```

## 属性

- `title` - 在警报规则中显示的操作标题
- `settings` - 警报规则设置的配置
- `uri` - 接收警报的端点
- `required_fields` - 必需字段列表
- `optional_fields` - 可选字段列表
