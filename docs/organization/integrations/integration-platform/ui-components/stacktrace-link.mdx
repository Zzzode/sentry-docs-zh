堆栈跟踪链接组件允许您在 Sentry 的堆栈跟踪中添加指向外部服务的链接。这些链接将出现在堆栈跟踪中的每一帧旁边。

## 架构

```json {filename:schema.json}
{
  "type": "stacktrace-link",
  "uri": "/sentry/stacktrace-link",
  "link": {
    "uri": "/sentry/link",
    "required_fields": [
      {
        "type": "text",
        "name": "line",
        "label": "Line"
      }
    ]
  }
}
```

## 属性

- `uri` - 接收请求的端点
- `link` - 链接配置
- `required_fields` - 必需字段列表
