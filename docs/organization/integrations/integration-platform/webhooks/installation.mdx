[公共 Sentry 集成](/organization/integrations/integration-platform/public-integration)将接收安装和卸载 webhooks。

## Sentry-Hook-Resource 标头

- `'Sentry-Hook-Resource': 'installation'`

## 属性

### action

- 类型：字符串
- 描述：`created` 或 `deleted`

### actor

- 类型：对象
- 描述：安装或卸载集成的用户

### data['installation']

- 类型：对象
- 描述：安装
