## Sentry-Hook-Resource 标头

`'Sentry-Hook-Resource': 'metric_alert'`

## 属性

### action

- 类型：字符串
- 描述：将是以下之一（`critical`、`warning`、`resolved`）

### data['metric_alert']

- 类型：对象
- 描述：触发警报规则的事件

### data['metric_alert']['alert_rule']

- 类型：对象
- 描述：警报规则

### data['metric_alert']['alert_rule']['triggers']

- 类型：对象数组
- 描述：触发警报的阈值

### data['metric_alert']['alert_rule']['triggers'][0]['actions']

- 类型：对象数组
- 描述：当触发器的阈值达到时将执行的操作

### data['description_text']

- 类型：字符串
- 描述：警报的人类可读描述

### data['description_title']

- 类型：字符串
- 描述：警报的人类可读标题

### data['web_url']

- 类型：字符串
- 描述：事件的 api url

<Alert>

以下字段仅用于[警报规则操作 UI 组件](/organization/integrations/integration-platform/ui-components/alert-rule-action)。

</Alert>

### data['metric_alert']['alert_rule']['triggers'][0]['actions'][0]['settings']

- 类型：对象
- 描述：在外部服务中路由警报的已保存配置
