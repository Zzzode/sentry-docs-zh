Sentry 与您喜欢的应用程序和服务集成。每个集成都提供有助于跟踪和分类错误的功能。如果您在对 Sentry 实例或组织进行更改后遇到某些集成问题，请查看我们的 [故障排除指南](/organization/integrations/troubleshooting/)。

## 集成平台

Sentry 的集成平台提供了一种让外部服务与 Sentry 交互的方式。利用此平台的集成是 Sentry 中的一流参与者，您可以为 [公共](/organization/integrations/integration-platform/public-integration) 和 [内部](/organization/integrations/integration-platform/internal-integration) 用例构建它们。

有关更多详细信息，请参阅 [完整集成平台文档](/organization/integrations/integration-platform/)。

## 可切换功能

某些集成具有可以通过集成级别的切换开关启用或禁用的功能。切换功能开/关会在整个集成中立即生效。默认值取决于集成的功能。

并非每个集成都有可切换功能，但如果它们有，您可以在特定集成的设置的 `功能` 选项卡中找到它们。

![集成功能选项卡](./img/integration-features-tab.png)
