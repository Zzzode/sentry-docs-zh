Linear 是一个为速度而构建的问题跟踪工具，简化软件项目、任务和错误跟踪。轻松地从 Sentry 创建新的 Linear 问题，并将 Sentry 问题链接到现有的 Linear 问题。

此集成由创建它的公司维护和支持。更多详细信息，请查看我们的[集成平台文档](/organization/integrations/integration-platform/)。

## 安装和配置

<Alert>

需要 Sentry 所有者、管理员或管理员权限才能安装此集成。

Linear **不**适用于自托管 Sentry。

</Alert>

1. 导航到**设置 > 集成 > Linear**

   ![安装 Linear 集成](./img/linear-install-pg.png)

2. 按照完整的 [Linear 安装说明](https://linear.app/docs/sentry)进行操作。

## 问题管理

问题跟踪允许您从 Sentry 内创建 Linear 问题，并将 Sentry 问题链接到现有的 Linear 问题。问题管理可以通过两种方式配置——自动或手动。

### 自动

要自动配置问题管理，请创建[问题警报](/product/alerts/alert-types/#issue-alerts)或[指标警报](/product/alerts/alert-types/#metric-alerts)。在选择[操作](/product/alerts/create-alerts/issue-alert-config/#then-conditions-actions)时，对于问题警报选择"通知集成"然后选择"Linear"，或者对于指标警报从操作下拉菜单中选择"Linear"：

![创建 Linear 问题作为指标警报规则操作](./img/linear-rule.png)

当警报触发时将自动创建 Linear 问题。

### 手动

要手动配置问题管理，请导航到特定的 Sentry 问题，并在右侧面板中找到"链接问题"部分：

![从 Sentry 问题链接 Linear 问题](./img/linear-link-issue.png)

在这里，您将能够创建或链接 Linear 问题。

![创建 Linear 问题表单](./img/linear-create-issue.png)
