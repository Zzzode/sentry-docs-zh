错误、功能以及您在 Shortcut 中跟踪的任何其他内容现在可以并存。新的 Shortcut 集成功能与 Shortcut 插件具有功能对等性。如果您在两者之间进行选择，我们推荐安装 Shortcut 集成。

此集成由创建它的公司维护和支持。更多详细信息，请查看我们的[集成平台文档](/organization/integrations/integration-platform/)。

## 安装和配置

<Alert>

需要 Sentry 所有者、管理员或管理员权限才能安装此集成。

Shortcut **不**适用于自托管 Sentry。

</Alert>

1. 导航到**设置 > 集成 > Shortcut**

![安装 Shortcut 集成](./img/shortcut.png)

2. 按照完整的 [Shortcut 安装说明](https://help.shortcut.com/hc/en-us/articles/360026979272-Setting-up-the-Sentry-Integration)进行操作。
