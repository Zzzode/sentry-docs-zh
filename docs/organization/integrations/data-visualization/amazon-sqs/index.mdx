<Alert>

Amazon SQS集成已弃用，无法在新项目中安装。

</Alert>
Amazon SQS对于更深入的异常分析很有用，使您能够快速轻松地将异常返回到您自己的系统中。或者，用它来赋能其他团队，如商业智能功能。

此集成需要为您希望使用它的每个项目进行设置。它由[Sentry社区](https://open.sentry.io/)维护和支持。

## 安装和配置

<Alert>

需要Sentry所有者、管理员或管理员权限才能安装此集成。

</Alert>

导航到**设置 > 集成 > Amazon SQS**

![安装Amazon SQS集成](./img/amazon-sqs.png)

### 数据转发

在**[项目] > 设置 > 数据转发**中配置[数据转发](/concepts/data-management/data-forwarding/)，并为给定集成提供所需信息。

Amazon的负载与我们的标准API事件负载相同，并将随时间演变。有关此数据格式的更多详细信息，请查看我们的[API文档](/api/events/retrieve-an-event-for-a-project/)。
