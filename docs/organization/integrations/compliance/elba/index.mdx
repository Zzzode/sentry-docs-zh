此集成将 Sentry 与 Elba 的访问审查平台连接起来，为组织提供管理和审查 Sentry 内用户访问权限的综合解决方案。
这通过以下方式帮助用户增强安全性、确保合规性并提高运营效率：

- 风险降低：定期访问审查有助于识别和消除不必要或未经授权的访问，降低数据泄露和内部威胁的风险。
- 主动管理：早期发现权限异常允许及时采取纠正措施。

它连接到 Sentry 的 API 以获取有关组织成员的详细信息，包括他们的角色、团队成员资格和权限级别。检索到的数据在 Elba 的平台中呈现，管理员可以在其中系统地审查每个用户的访问权限。此外，该平台分析用户权限以识别潜在的安全风险，例如过度权限或过时的访问，并提供补救建议。

此集成由 Elba 维护和支持。有关更多详细信息或问题，请随时联系 support@elba.security。

## 安装和配置

<Alert>

需要 Sentry 所有者、管理员或拥有者权限才能安装此集成。

</Alert>

1. 导航到 **设置 > 集成 > Elba**

2. 遵循完整的 [Elba 安装说明](https://sentry.elba.security/install)。
