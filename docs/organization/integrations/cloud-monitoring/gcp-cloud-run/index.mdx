Cloud Run 帮助开发者节省构建和部署应用程序的时间。Sentry 通过提供详细的调试功能，帮助节省解决生产环境问题的时间。

## 安装和配置

<Alert>

需要 Sentry 所有者、管理员或拥有者权限才能安装此集成。

</Alert>

1. [创建账户](https://sentry.io/signup/)（或[登录](https://sentry.io/auth/login/)）Sentry

2. 选择您的应用程序平台

   ![Sentry 选择平台](./img/sentry-choose-platform.png)

3. 按照说明为您的 Cloud Run 应用程序进行埋点。详细的平台文档[在此](/platforms/)。

4. 使用 GCP 部署您的 Cloud Run 应用程序。
