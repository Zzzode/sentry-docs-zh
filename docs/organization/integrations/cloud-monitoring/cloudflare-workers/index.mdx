Cloudflare Workers 允许您在全球范围内部署无服务器应用程序，具有卓越的性能、可靠性和扩展性。

此集成由创建它的公司维护和支持。如需更多详细信息或问题，请通过其支持门户联系 Cloudflare：https://developers.cloudflare.com/support/contacting-cloudflare-support/

## 安装和配置

<Alert>

需要 Sentry 所有者、管理员或拥有者权限才能安装此集成。

</Alert>

1. 导航到 **设置 > 集成 > Cloudflare Workers**

2. 遵循完整的 [Cloudflare Workers 安装说明](https://developers.cloudflare.com/workers/observability/sentry-integration/)。
