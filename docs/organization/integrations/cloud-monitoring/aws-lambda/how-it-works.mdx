## Node

当 Sentry 添加到 Lambda 函数时，会对您的 Lambda 函数进行以下修改：

- Sentry 的层被添加到您的 Lambda 中。任何现有层都会被保留，Sentry 层会被追加到末尾。

  ![](./img/layer.png)

- Sentry 添加以下环境变量

  - `NODE_OPTIONS`：这是为了预加载 `awslambda-auto` 模块，它将自动初始化 Sentry
  - `SENTRY_DSN`：这被设置为您的项目的 [DSN](/product/sentry-basics/dsn-explainer/)
  - `SENTRY_TRACES_SAMPLE_RATE`：这为事务设置[采样率](/platforms/javascript/guides/node/configuration/sampling/#sampling-transaction-events)。如果您想要不同的采样率，可以手动编辑环境变量。

  ![](./img/env_variables.png)

## Python

当 Sentry 添加到 Lambda 函数时，会对您的 Lambda 函数进行以下修改：

- Sentry 的层被添加到您的 Lambda 中。任何现有层都会被保留，Sentry 层会被追加到末尾。

  ![](./img/layer_python.png)

  并且您的运行时处理程序将指向 Sentry 的处理程序，该处理程序将为您初始化 SDK，然后调用您的运行时处理程序来调用您的函数

  ![](./img/handler_python.png)

- Sentry 添加以下环境变量

  - `SENTRY_INITIAL_HANDLER`：这存储您的运行时处理程序的值。如果您选择禁用 Sentry 层，将使用此值来设置运行时处理程序。
  - `SENTRY_DSN`：这被设置为您的项目的 [DSN](/product/sentry-basics/dsn-explainer/)
  - `SENTRY_TRACES_SAMPLE_RATE`：这为事务设置[采样率](/platforms/python/configuration/sampling/#sampling-transaction-events)。如果您想要不同的采样率，可以手动编辑环境变量。

  ![](./img/env_variables_python.png)
