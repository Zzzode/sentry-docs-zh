如果您更改了组织标识符，您需要更新此集成的配置。在我们的[故障排除指南](/organization/integrations/troubleshooting)中了解更多信息。

</Alert>

Split 集成可以快速处理并在 Split 平台中将 Sentry 异常数据显示为跟踪事件以供分析。您可以在 Split 仪表板中控制为哪些环境和流量类型捕获异常，而无需触及任何代码。

此集成由创建它的公司维护和支持。更多详细信息，请查看我们的[集成平台文档](/organization/integrations/integration-platform/)。

## 安装和配置

<Alert>

需要 Sentry 所有者、管理员或管理员权限才能安装此集成。Split 集成仅适用于商业和企业计划的组织。

Split **不**适用于自托管 Sentry。

</Alert>

1. 导航到**设置 > 集成 > Split**

![安装 Split 集成](./img/split.png)

2. 然后，按照完整的 [Split 安装说明](https://help.split.io/hc/en-us/articles/360029879431)进行操作。
