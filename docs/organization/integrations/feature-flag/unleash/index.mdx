## 评估跟踪

Sentry 可以跟踪应用程序中发生的标志评估。标志评估将显示在问题详情页面的"功能标志"部分中，以表格形式呈现，其中"可疑"标志预测会以黄色突出显示。通过阅读[问题详情页面文档](/product/issues/issue-details/#feature-flags)了解如何在 Sentry UI 中与功能标志洞察进行交互。

### 设置评估跟踪

要设置评估跟踪，请访问我们支持的某个语言页面：
* [JavaScript](/platforms/javascript/configuration/integrations/unleash/)
* [Python](/platforms/python/integrations/feature-flags/unleash/)

## 变更跟踪

Sentry 可以跟踪功能标志定义的更改并报告可疑的功能标志编辑。

### 设置变更跟踪

启用变更跟踪是一个四步过程。要开始，请在新标签页中访问[功能标志设置页面](https://sentry.io/orgredirect/organizations/:orgslug/settings/feature-flags/change-tracking/)。然后按照下面列出的步骤操作。

1. **点击"添加新提供商"按钮。**
    - 每种提供商类型可以注册一个 webhook 密钥。
    - 在下拉菜单中选择 Unleash，该下拉菜单显示"选择提供商"。
2. **注册 webhook URL。**
    - 转到您的 Unleash 主页并导航到 `/integrations/` 页面，可以通过点击左侧边栏导航中的集成，在配置标题下找到该页面。
    - 选择 Webhook 选项。您应该位于 `/integrations/create/webhook/` 页面。
    - 从 Sentry 设置中复制 webhook URL 并将其粘贴到 Unleash 的 webhook 集成 UI 中。
    - 确保集成已切换为启用。
3. **设置签名密钥。**
    - 在 Unleash webhook UI 中，在授权输入框下，输入任意 32 个字符的字符串作为您的授权令牌（"密钥"）。请妥善保管此字符串，并将其记在某处以备将来需要时使用；Unleash 在保存 webhook 后不会向您显示令牌。我们建议不要使用任何敏感令牌。
    - 从 Unleash UI 复制您的授权令牌并将其粘贴到 Sentry 设置中"密钥"旁边的输入框中。
4. **配置您的 webhook。**
    - 在事件输入框下，指定您希望发送到 Sentry 的功能标志事件。我们目前支持以下事件：![支持的 Unleash 事件](./img/unleash-events.png)
    - 通过点击 Sentry 设置中的"添加提供商"来保存密钥。
    - 通过点击 Unleash 中的"创建"来保存 webhook。

保存后，Sentry 现在将接受并验证发送到您组织的功能标志 webhook 端点的所有入站 hook。
