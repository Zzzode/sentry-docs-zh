## 评估跟踪

Sentry 可以跟踪应用程序中发生的标志评估。标志评估将显示在问题详情页面的"功能标志"部分中，以表格形式呈现，其中"可疑"标志预测会以黄色突出显示。通过阅读[问题详情页面文档](/product/issues/issue-details/#feature-flags)了解如何在 Sentry UI 中与功能标志洞察进行交互。

### 设置评估跟踪

Flagsmith 建议使用 OpenFeature SDK 结合 [Flagsmith 提供商](https://openfeature.dev/ecosystem?instant_search%5Bquery%5D=flagsmith&instant_search%5BrefinementList%5D%5Btype%5D%5B0%5D=Provider)来评估功能标志。要跟踪这些评估，请访问我们支持的某个语言页面以获取 OpenFeature：
* [JavaScript](/platforms/javascript/configuration/integrations/openfeature/)
* [Python](/platforms/python/integrations/openfeature/)

访问 [Flagsmith 文档](https://docs.flagsmith.com/integrations/apm/sentry#evaluation-tracking-setup)获取有关如何设置带有 Flagsmith 提供商的 OpenFeature SDK 的具体说明。

## 变更跟踪

Sentry 可以跟踪功能标志定义的更改并报告可疑的功能标志编辑。

### 设置变更跟踪

启用变更跟踪是一个三步过程。要开始，请在新标签页中访问[功能标志设置页面](https://sentry.io/orgredirect/organizations/:orgslug/settings/feature-flags/change-tracking/)。然后按照下面列出的步骤操作。

1. **点击"添加新提供商"按钮。**
    - 每种提供商类型可以注册一个 webhook 密钥。
    - 在下拉菜单中选择 Generic，该下拉菜单显示"选择提供商"。
2. **注册 webhook URL。**
    ![Flagsmith Sentry Webhook 模态框](./img/flagsmith-sentry-webhook-modal.png)
    - 转到您的 Flagsmith 项目仪表板并导航到 `/project/:projectid/integrations` 页面，可以通过点击顶部导航栏中的"集成"找到该页面。
    - 找到 Sentry 选项，然后点击"添加集成"，这将打开一个模态框。
    - 选择将触发标志更改事件的环境。
    - 从 Sentry 设置中复制 webhook URL 并将其粘贴到 Flagsmith "Sentry 集成"模态框中的"Webhook URL"。
3. **设置签名密钥。**
    - 仍在 Flagsmith "Sentry 集成"模态框中，输入任意 10 到 60 个字符之间的字符串作为您的授权令牌（"密钥"）。
    - 从上一步复制授权令牌并将其粘贴到 Sentry 设置中"密钥"旁边的输入框中。

保存后，Sentry 现在将接受并验证发送到您组织的功能标志 webhook 端点的所有入站 hook。
