如果您有兴趣成为早期采用者，您可以在 **设置 > 常规设置** 中打开/关闭您组织的早期采用者状态。这将影响您组织中的所有用户，并且可以同样轻松地关闭。

![设置中启用的早期采用者开关。](./img/early-adopter-toggle.png)

此页面列出了当您选择加入"早期采用者"时将获得访问权限的功能。请注意，功能有时会分波向早期采用者发布，因此启用"早期采用者"设置后，您可能不会立即看到功能。

限制：

- 此列表不包括不受"早期采用者"设置控制的新功能，例如 alpha、封闭测试版或需要手动选择加入的有限可用性功能。
- 此列表不能保证 100% 最新，但会频繁监控和更新。
