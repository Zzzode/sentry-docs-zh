此设置假设您已安装[Yarn][Yarn]、[Volta][volta]和[pre-commit][pre-commit]。

如果您不想使用Volta，请确保使用Node 20。

首先，打开终端并克隆`sentry-docs`仓库：

```bash
# 克隆文档仓库
git clone git@github.com:getsentry/sentry-docs.git
```

接下来，进入克隆的仓库并运行以下命令安装依赖：

```bash
# 安装或更新应用程序依赖
make
```

现在，运行开发Web服务器（取决于您要运行的文档）：

```bash
# 为用户文档启动开发服务器
yarn dev

# 为开发者文档启动开发服务器
yarn dev:developer-docs
```

您现在可以通过http://localhost:3000访问文档。

[Next.js]: https://nextjs.org/
[volta]: https://volta.sh/
[pre-commit]: https://pre-commit.com/
[Yarn]: https://yarnpkg.com/

## 代码检查

有几个代码检查工具可用。运行`yarn lint`应用所有检查。

一些检查可以通过eslint自动修复

```bash
yarn lint:eslint:fix
```

此外，我们使用prettier格式化代码和markdown。如果在CI中获得代码检查错误，请运行prettier：

```bash
yarn lint:prettier:fix
```
