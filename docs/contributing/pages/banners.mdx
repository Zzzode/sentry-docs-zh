您可以通过在 `banner/index.tsx` 文件的 `BANNERS` 数组中添加条目来在页面顶部添加任意横幅。`BANNERS` 数组是具有以下属性的对象数组：

```typescript {filename:banner/index.tsx}
type BannerType = {
  /** 这是一个字符串或正则表达式数组，用于输入 new RegExp() */
  appearsOn: (string | RegExp)[];
  /** 行动号召按钮的标签 */
  linkText: string;
  /** 行动号召按钮的目标URL */
  linkURL: string;
  /** 横幅的主要文本 */
  text: string;
  /** 可选的ISO日期字符串，将在该日期后隐藏横幅，无需重建 */
  expiresOn?: string;
};
```

您可以根据需要添加任意数量的横幅。如果您需要禁用所有横幅，只需将它们从数组中删除即可。

每个横幅按顺序评估，第一个匹配的将显示。

示例：

```typescript {filename:banner/index.tsx}
// ...
// appearsOn = [];              // 这是禁用的
// appearsOn = ['^/$'];         // 这在主页上启用
// appearsOn = ['^/welcome/'];  // 这在 "/welcome" 页面上启用
// ...

const BANNERS = [
  // 这个将优先于数组中的最后一个横幅
  // （匹配所有 /platforms 页面），因为它先匹配。
  {
    appearsOn: ['^/platforms/javascript/guides/astro/'],
    text: '此横幅出现在Astro指南上',
    linkURL: 'https://sentry.io/thought-leadership',
    linkText: '获取网络研讨会',
  },

  // 这个将匹配 /welcome 页面和所有 /for 页面
  {
    appearsOn: ['^/$', '^/platforms/'],
    text: '此横幅出现在主页和所有 /platforms 页面上',
    linkURL: 'https://sentry.io/thought-leadership',
    linkText: '获取网络研讨会',
  },
];

```

可选地，您可以向横幅添加 `expiresOn` 属性，以在某个日期后隐藏它，而无需重建或手动删除。
ISO日期字符串应采用 `YYYY-MM-DDTHH:MM:SSZ` 格式以正确解析并考虑时区。

```typescript {filename:banner/index.tsx}
const BANNERS = [
  {
    appearsOn: ['^/$'],
    text: '此主页横幅将在2024-12-06后消失',
    linkURL: 'https://sentry.io/party',
    linkText: 'RSVP',
    expiresOn: '2024-12-06T00:00:00Z',
  },
];
```
