跨度是 Sentry SDK 中捕获的底层数据元素，它们共同构成一个跟踪。

您可以在以下 [Sentry](https://sentry.io) 页面中按跨度属性进行搜索：

- [跟踪](/product/explore/traces/) - 当搜索包含具有匹配属性的跨度的跟踪时

当您在**跟踪**页面中搜索跨度属性时，搜索将返回包含_一个或多个匹配跨度_的任何跟踪。

## 可搜索属性

以下是可用于跨度搜索的键和标记列表：

<DefinitionList>

### `action`

跨度操作的类型，例如，SQL 跨度的 `SELECT` 或 HTTP 跨度的 `POST`。

- **类型：** 字符串

### `browser.name`

生成跨度的浏览器名称。

- **类型：** 字符串

### `cache.hit`

缓存读取是否命中或错过了查询的缓存键。值为 `'true'` 或 `'false'`。

- **类型：** 字符串

### `description`

跨度的参数化和清理描述。

- **类型：** 字符串

### `device.class`

设备类别是一个合成字段，通过使用上下文中找到的设备信息（如 iOS 设备的型号）和设备规格（如 Android 设备的 `processor_frequency`）来计算。

- **类型：** 字符串

### `domain`

跨度操作的通用范围，例如，`db` 跨度中涉及的表或 `http` 跨度中的主机名。

- **类型：** 字符串

### `duration`

跨度所花费的总时间。

- **类型：** 持续时间

### `environment`

指您的代码部署命名约定。例如，`development`、`testing`、`staging` 等。[了解更多](/product/sentry-basics/environments/)。在 [Sentry](https://sentry.io/) 的某些页面中，您可以使用下拉菜单按环境过滤。

- **类型：** 字符串

### `file_extension`

资源跨度的文件扩展名。

- **类型：** 字符串

### `group`

跨度描述的唯一哈希。

- **类型：** 字符串

### `http.decoded_response_content_length`

资源的[解码主体大小](https://developer.mozilla.org/en-US/docs/Web/API/PerformanceResourceTiming/decodedBodySize)。

- **类型：** 字符串

### `http.response_content_length`

资源的[编码主体大小](https://developer.mozilla.org/en-US/docs/Web/API/PerformanceResourceTiming/encodedBodySize)。

- **类型：** 字符串

### `http.response_transfer_size`

资源的[总传输大小](https://developer.mozilla.org/en-US/docs/Web/API/PerformanceResourceTiming/transferSize)。

- **类型：** 字符串

### `messaging.destination.name`

消息写入或发布到的队列名称或主题。消费者也从该队列读取。

- **类型：** 字符串

### `messaging.message.id`

来自队列或消息系统的消息的唯一 ID。

- **类型：** 字符串

### `module`

跨度关联的洞察模块，例如，`cache`、`db`、`http` 等。

- **类型：** 字符串

### `op`

跨度的操作，例如，`http.client`、`middleware`。

- **类型：** 字符串

### `os.name`

操作系统的名称。

- **类型：** 字符串

### `platform`

平台名称。这默认为 `other`，仅适用于此列表中的平台：
`as3`、`c`、`cfml`、`cocoa`、`csharp`、`elixir`、`go`、`groovy`、`haskell`、`java`、`javascript`、`native`、`node`、`objc`、`other`、`perl`、`php`、`python`、`ruby`

- **类型：** 字符串

### `release`

发布是部署到环境的代码版本。您可以创建与发布完全匹配的标记，或使用 `release:latest` 选择最近的发布。[了解更多](/product/releases/usage/sorting-filtering/#latest-release)。

- **类型：** 字符串

### `resource.render_blocking_status`

资源的[渲染阻塞状态](https://developer.mozilla.org/en-US/docs/Web/API/PerformanceResourceTiming/renderBlockingStatus)。

- **类型：** 字符串

### `sdk.name`

发送事件的 Sentry SDK 的名称。

- **类型：** 字符串

### `sdk.version`

发送事件的 Sentry SDK 的版本。

- **类型：** 字符串

### `self_time`

跨度的持续时间，不包括其子跨度的持续时间。

- **类型：** 字符串

### `status`

跨度所代表操作的状态。

### `status_code`

HTTP 响应状态代码。

- **类型：** 字符串

### `system`

数据库系统（例如，`postgresql` 或 `mysql`）。

- **类型：** 字符串

### `trace.status`

跨度跟踪的成功或失败状态。

- **类型：** 字符串

### `transaction`

包含事务的名称。

- **类型：** 字符串


### `transaction.method`

包含事务的 HTTP 方法。

- **类型：** 字符串

### `transaction.op`

包含事务的操作。

- **类型：** 字符串

### `user.email`

用户名的替代或补充。Sentry 了解电子邮件地址，因此可以显示 Gravatars 等功能并解锁消息传递功能。

- **类型：** 字符串

### `user.id`

用户的应用程序特定内部标识符。

- **类型：** 字符串

### `user.username`

用户名，通常比 `user.id` 更好的标签。

- **类型：** 字符串

</DefinitionList>
