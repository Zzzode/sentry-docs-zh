Codecov，面向开发者的测试代码覆盖率领先提供商，现在是 Sentry 家族的一部分。它为二十多个测试框架提供可操作的覆盖率，并帮助改进测试、覆盖率和代码可靠性。使用 Codecov，您将能够直接在 Sentry 中获取测试覆盖率见解，同时监控您的错误。在此处阅读有关 Codecov 的更多信息[此处](https://docs.codecov.com/docs/about-code-coverage)。
