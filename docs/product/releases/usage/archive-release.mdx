由于从已删除发布接收到的事件会自动重新创建它，Sentry提供了一个隐藏或归档发布的选项。如果您不再希望查看发布，或者您意外创建了不再希望跟踪的发布，归档也很有用。

## 使用Sentry.io

在[sentry.io](https://sentry.io)中归档会从您可以选择查看的发布列表中删除发布：

![归档发布界面](./img/release-archive.png)

归档的发布不会显示在**发布**中，除非您更改过滤器以显示它们：

![归档发布界面](./img/releases-filter-archived.png)

## 使用CLI

在CLI中，运行此命令以归档发布：

```bash
sentry-cli releases archive VERSION
```
