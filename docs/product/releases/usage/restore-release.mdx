您可以使用[sentry.io](https://sentry.io)或Sentry CLI恢复已归档的发布。

## 使用Sentry.io

在[sentry.io](https://sentry.io)中，您可以通过点击页面顶部的上下文菜单并选择"恢复"来恢复发布：

![恢复发布界面](./img/release-restore.png)

## 使用CLI

在CLI中，通过运行此命令恢复发布：

```bash
sentry-cli releases restore VERSION
```
