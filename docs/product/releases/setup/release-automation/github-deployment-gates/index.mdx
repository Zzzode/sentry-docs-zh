此集成已弃用，将来也将为现有组织移除。

</Alert>

[Sentry GitHub 部署门禁](https://github.com/apps/sentry-deployment-gate) 允许 Sentry 通过部署保护规则与 GitHub 发布进行交互。发送 Sentry 发布信息后，您可以监视发布中的错误，并在完全部署之前拒绝金丝雀发布。

有关 Sentry 部署门禁的更多详细信息，请参阅 [GitHub 应用程序](https://github.com/apps/sentry-deployment-gate)。

<Alert level="warning">

如果您更改了组织标识符，您需要更新此集成的配置。在我们的[故障排除指南](/organization/integrations/troubleshooting)中了解更多信息。

</Alert>
