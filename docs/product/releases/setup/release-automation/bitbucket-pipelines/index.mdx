<Alert level="warning">

如果您更改了组织标识符，您需要更新此集成的配置。在我们的[故障排除指南](/organization/integrations/troubleshooting)中了解更多信息。

</Alert>

[Sentry Release Pipe](https://bitbucket.org/sentryio/sentry-new-release) 只需一步即可在 Bitbucket Pipelines 中自动化 Sentry 发布管理。发送 Sentry 发布信息后，您将能够将提交与发布关联起来。您还能够应用源映射以在 Sentry 中查看原始代码。

有关 Sentry 发布管理概念的更多详细信息，请参阅[发布](/product/releases/)的完整文档。
