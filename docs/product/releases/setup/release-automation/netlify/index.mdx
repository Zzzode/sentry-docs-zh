Sentry Netlify 构建插件在 Netlify 中自动化 Sentry 发布管理。设置完成后，它会自动将发布和提交信息发送到 Sentry，并上传源映射，使您能够在堆栈跟踪中看到原始代码。

完整的文档可在[插件的 README](https://github.com/getsentry/sentry-netlify-build-plugin) 中找到。

有关 Sentry 发布管理概念的更多详细信息，请参阅我们的[发布](/product/releases/)文档。

<Alert>

注意：此构建插件与 `@netlify/sentry` 是分开的，后者是由 Netlify 构建的监控插件。该插件的文档可以在[这里](https://docs.netlify.com/netlify-labs/experimental-features/sentry-integration/)找到。有关更多信息，请参阅 [`@sentry/netlify-build-plugin` 与 `@netlify/sentry`](https://github.com/getsentry/sentry-netlify-build-plugin#sentrynetlify-build-plugin-vs-netlifysentry)。

</Alert>
