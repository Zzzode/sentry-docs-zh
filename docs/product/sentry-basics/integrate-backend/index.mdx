这个逐步教程将指导您完成设置和配置 Sentry 账户以监控 Python 后端应用程序中的错误的流程。作为本教程的一部分，您将：

- 向 Sentry SDK 添加基本配置
- 将各种类型的事件和错误监控到 Sentry
- 在您的应用程序前端和后端之间实现错误追踪

本教程使用一个[示例 Django 应用程序](https://github.com/sentry-tutorials/backend-monitoring)。这使我们能够简化将 Sentry 集成到您的开发工作流程中所需的步骤，以获得最大价值。

或者，您可以将本指南作为参考，并直接将相关更改应用到您的源代码中。

## 先决条件

- 一个 [Sentry](https://sentry.io) 账户，或[注册](https://sentry.io/signup/)一个
- 一个 [GitHub](https://github.com/) 账户，或[注册](https://github.com/join)一个
