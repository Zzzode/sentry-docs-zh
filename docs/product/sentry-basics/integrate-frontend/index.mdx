这个逐步教程将指导您在 React 项目中设置 Sentry 以监控前端应用程序中的错误。完成本教程后，您将能够：

- 在代码发生错误时接收警报
- 查看带有源代码上下文的可读堆栈跟踪
- 根据提交信息和堆栈跟踪查看可疑提交和建议的负责人

本教程使用一个[示例 React 应用程序](https://github.com/getsentry/frontend-tutorial)。您可以跟随提供的示例代码操作，或将相同的步骤应用到您自己的项目中。

## 先决条件

- 一个 [Sentry](https://sentry.io) 账户，或[注册](https://sentry.io/signup/)一个
- 一个 [GitHub](https://github.com/) 账户，或[注册](https://github.com/join)一个
- [Node 18+](https://nodejs.org/en/download/)
