title: "会话重放"
sidebar_order: 70
description: "使用会话重放获取用户会话的重现，以改善您的应用体验。"
---

Sentry 支持 Web 和移动端的会话重放。这包括基于浏览器的应用程序和某些原生移动平台，如 Android、iOS 和 React Native。重放的两个版本（Web 和移动）都已正式发布且稳定。

对于基于浏览器的应用程序，这包括静态网站、单页应用程序，也包括服务器端渲染的应用程序——例如，Electron、Next.js 和 Remix 等平台。

要了解我们支持哪些 SDK，请访问我们的 [Web](web/#supported-sdks) 和 [移动](mobile/#supported-sdks) 文档。
