会话重放允许您查看用户会话的视频式重现，这可以帮助您了解错误发生之前、期间和之后发生了什么。您将能够获得更深入的调试上下文，以便在没有猜测的情况下更快地重现和解决问题。当您回放每个会话时，您将能够看到每个用户交互与网络请求、DOM 事件和控制台消息的关系。这实际上就像在您的生产用户会话中激活了 [DevTools](https://developer.chrome.com/docs/devtools/overview/)。

重放与 Sentry 产品的其他部分集成，因此您可以看到用户体验如何受到错误和慢事务的影响。您将在[问题详情](/product/issues/issue-details/)页面上看到与错误事件关联的会话重放，以及在[事务摘要](/product/insights/overview/transaction-summary/)页面上看到与慢事务关联的会话重放。对于[后端错误重放](/product/explore/session-replay/web/getting-started/#replays-for-backend-errors)，任何相关的后端错误都将包含在重放的时间线、[面包屑](/product/issues/issue-details/breadcrumbs/)和错误中。

## 什么是会话重放？

会话重放**不是**视频录制。它是用户会话的视频式重现，使用 [rrweb 录制库](https://www.rrweb.io/) 构建。重放是从 Web 应用程序的 DOM 状态快照（浏览器的内存中 HTML 表示）创建的。当每个快照被回放时，您将看到用户在他们的整个会话中所做操作的视频式录制。用户会话包括用户访问您的网站时发生的任何页面加载、刷新和导航。

![会话重放用户界面](./img/session-replay-what-is-a-replay.png)

会话重放录制的开始可以通过以下方式触发：

- 用户会话成为采样率的一部分，由 [`replaysSessionSampleRate`](/platforms/javascript/session-replay/#sampling) 控制。（当用户加载页面时，决定是否对其进行采样。）
- 在未被记录的会话中发生错误。然后基于 [`replaysOnErrorSampleRate`](/platforms/javascript/session-replay/#sampling) 记录会话。
- 手动调用 [`replay.start()`](/platforms/javascript/session-replay/understanding-sessions/#manually-starting-replay) 方法。

会话重放录制的结束可以通过以下方式触发：

- 在被记录的标签页或页面中的用户不活动。（当用户在 15 分钟内没有点击或在网站周围导航时，被视为不活动。鼠标滚动、鼠标移动和键盘事件目前不被视为活动。）
- 用户关闭被记录的标签页或页面。
- 录制达到最大重放持续时间限制。（目前，这是 60 分钟。）
- 手动调用 [`replay.stop()`](/platforms/javascript/session-replay/understanding-sessions/#manually-stopping-replay) 方法。

<Alert>

与**发布**页面上的[会话](/product/releases/health/#session)不同，会话重放中的用户会话可以跨越多个页面加载。

</Alert>

## 支持的 SDK

会话重放支持所有基于浏览器的应用程序。
这包括静态网站、单页应用程序，也包括服务器端渲染。这包括以下框架：
[Django](/platforms/python/integrations/django/)、
[Spring](/platforms/java/guides/spring-boot/)、
[ASP.NET](/platforms/dotnet/guides/aspnetcore/)、
[Laravel](/platforms/php/guides/laravel/)、
[Express](/platforms/javascript/guides/express/) 和
[Rails](/platforms/ruby/guides/rails/)。
如果您不使用 `npm` 或 `yarn`，您可以在主 HTML 模板上使用[我们的加载器 `script` 标签](/platforms/javascript/install/loader/)。

录制重放的 Sentry SDK 在客户端浏览器上运行，它内置于 `@sentry/browser` 和我们的浏览器框架 SDK 中：

- [Vanilla JavaScript](/platforms/javascript/session-replay/)
- [Angular](/platforms/javascript/guides/angular/session-replay/)
- [Astro](/platforms/javascript/guides/astro/session-replay/)
- [Capacitor](/platforms/javascript/guides/capacitor/session-replay/)
- [Electron](/platforms/javascript/guides/electron/session-replay/)
- [Ember](/platforms/javascript/guides/ember/session-replay/)
- [Gatsby](/platforms/javascript/guides/gatsby/session-replay/)
- [Next.js](/platforms/javascript/guides/nextjs/session-replay/)
- [React](/platforms/javascript/guides/react/session-replay/)
- [Remix](/platforms/javascript/guides/remix/session-replay/)
- [Svelte](/platforms/javascript/guides/svelte/session-replay/)
- [SvelteKit](/platforms/javascript/guides/sveltekit/session-replay/)
- [Vue](/platforms/javascript/guides/vue/session-replay/)
