# 合作伙伴平台

Sentry 提供强大的合作伙伴功能，允许您将错误监控和性能洞察直接集成到您的平台中。无论您是开发者平台、托管提供商还是 SaaS 应用程序，您都可以利用 Sentry 的监控功能来增强您的客户体验。

## 可用功能

Sentry 为合作伙伴提供了两种主要的集成路径：

### 1. [账户配置 API](/product/partnership-platform/account-provisioning-api/)

使用我们的配置 API 自动为您的客户创建 Sentry 组织。这允许您：

- **无缝引导客户**使用 Sentry 监控
- **创建预配置的组织**包含项目和团队
- **通过集成令牌管理客户访问权限**
- **支持多个区域**（美国和欧盟数据驻留）

### 2. [OAuth 集成](/product/partnership-platform/oauth-integration/)

通过 OAuth 让现有 Sentry 客户将其账户连接到您的平台：

- **您的平台和 Sentry 组织之间的账户链接**
- **具有可配置范围的基于权限的访问**
- **用户控制的权限**并易于撤销
- **为您的用户提供无缝的身份验证流程**

## 为您的平台带来的好处

与 Sentry 的合作伙伴平台集成提供了几个优势：

- **增强的开发者体验**内置错误监控
- **减少支持负担**帮助客户更快地调试问题
- **具有企业级监控的竞争差异化**
- **通过监控功能带来的收入机会**
- **通过更好的应用程序可靠性提高客户保留率**
