Sentry 的 Cron 监控允许您在 Sentry 中监控任何计划、周期性作业的正常运行时间和性能。一旦实施，它将允许您获得警报和指标，帮助您解决错误、检测超时并防止服务中断。

## Cron 监控用例示例

- 发票生成作业，其中每个作业运行成功完成且及时完成至关重要，具有预定义的、计划的间隔和频率。
- 服务器清理工具，其中每个作业执行不得超过预定义的运行时阈值。
- 任何其他具有有限输入的独立执行单元。

要使用 Cron 监控，您必须拥有现有的 Sentry 账户和项目设置。如果您没有，请 [在此处创建账户](https://sentry.io/signup/)。
