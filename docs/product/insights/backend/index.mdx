Sentry的[**后端性能**](https://sentry.io/orgredirect/organizations/:orgslug/insights/backend/)页面为您提供了应用程序健康状况的概览。您将能够看到**最耗时的查询**、**最耗时的域**、**p50**和**p75持续时间**等内容。

![后端性能概览页面。](../img/backend-performance-overview.png)

您还可以深入了解查询、出站API请求、缓存和队列，以获取有关影响应用程序健康状况的潜在问题的详细信息。除了拥有专门监控后端性能的空间外，您还可以查看Sentry的**洞察**标签页来监控前端、移动和AI性能。
