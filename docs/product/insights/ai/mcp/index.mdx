Sentry的MCP（模型上下文协议）监控工具帮助您了解MCP实现中发生的情况。它们自动收集有关MCP服务器连接、资源访问、工具执行和错误的信息，涵盖整个MCP管道——从客户端请求到服务器响应。

## MCP监控用例示例

- 您的MCP服务器未能响应工具调用，您想要跟踪完整的请求流程以识别连接中断的位置。
- 客户端报告您的MCP资源返回过时或格式错误的数据，您需要调试资源请求和服务器响应的完整上下文。
- 您的MCP实现遇到性能问题，您想要识别哪些组件（服务器启动、资源获取或工具执行）正在造成瓶颈。

## 开始使用

要使用MCP监控，您必须拥有现有的Sentry帐户和项目设置。如果您还没有，请[在此处创建帐户](https://sentry.io/signup/)。
