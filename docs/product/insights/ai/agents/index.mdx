Sentry的AI代理监控工具帮助您了解AI代理工作流的情况。它们自动收集有关代理运行、工具调用、模型交互和错误的信息，涵盖整个AI管道——从用户交互到最终响应。

## AI代理监控用例示例

- 您的AI代理在工具执行期间静默失败，您想要跟踪完整的代理流程以识别故障位置。
- 用户报告您的AI代理返回意外或格式错误的响应，您需要调试提示、模型调用和输出的完整上下文。
- 您的AI代理工作流遇到性能问题，您想要识别哪些步骤（模型调用、工具使用或自定义逻辑）正在造成瓶颈。
