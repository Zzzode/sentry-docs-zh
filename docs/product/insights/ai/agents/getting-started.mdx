Sentry AI代理监控帮助您使用我们支持的SDK和集成来跟踪和调试AI代理应用程序。从用户交互到最终响应，监控您的完整代理工作流，包括工具调用、模型交互和自定义逻辑。

要开始将AI代理数据发送到Sentry，请确保您已为启用AI的存储库创建了Sentry项目，并遵循以下指南之一：

## 支持的SDK

#### 使用Vercel AI SDK快速开始

```javascript
import * as Sentry from "@sentry/node";

// Sentry初始化需要在所有其他内容之上
Sentry.init({
  tracesSampleRate: 1.0,
  integrations: [Sentry.vercelAIIntegration()],
});

import { generateText } from "ai";
import { openai } from "@ai-sdk/openai";

// 您的AI代理函数
async function aiAgent(userQuery) {
  const result = await generateText({
    model: openai("gpt-4o"),
    prompt: userQuery,
    experimental_telemetry: {
      isEnabled: true,
      functionId: "ai-agent-main",
      recordInputs: true,
      recordOutputs: true,
    },
  });

  return result.text;
}
```

#### 使用OpenAI代理快速开始

```python
import sentry_sdk
import agents
from pydantic import BaseModel

sentry_sdk.init(
    dsn="YOUR_DSN",
    traces_sample_rate=1.0,
    send_default_pii=True,  # 包括LLM输入/输出
)

# 创建您的AI代理
my_agent = agents.Agent(
    name="My Agent",
    instructions="你是一个有用的助手。",
    model="gpt-4o-mini",
)

# 您的AI代理函数
result = await agents.Runner.run(
    my_agent,
    input=user_query,
)

```
