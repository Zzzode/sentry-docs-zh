如果您尚未启用性能监控和跟踪，请使用下面支持的SDK链接快速设置对我们[性能监控](/product/performance/)功能的访问。性能监控和跟踪帮助您从宏观级[指标](/product/performance/metrics/)到微观级[span](/concepts/key-terms/tracing/trace-view/#产品演练-跟踪视图页面)查看所有内容，您将能够[将事务与相关问题交叉引用](/product/performance/transaction-summary/)，根据您的个人需求自定义[查询](/product/explore/discover-queries/query-builder/)，以及更多。
