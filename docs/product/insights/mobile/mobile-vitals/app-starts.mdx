**应用程序启动**页面显示了应用程序完成冷启动和热启动所需时间的概览。它帮助您识别缓慢或回归的屏幕，并提供额外信息，以便您更好地理解影响应用程序启动时间缓慢的因素。

### 最低SDK要求：

**对于Android：**

- `>=7.4.0`用于应用程序启动span和应用程序启动分析的自动检测
- [性能-V2](/platforms/android/performance/instrumentation/perf-v2)功能标志（`<8.0.0`）和[应用程序启动分析](/platforms/android/profiling/#app-start-profiling)也必须在SDK中启用，例如：

```java {filename:MyApplication.java}
import io.sentry.android.core.SentryAndroid;

SentryAndroid.init(this, options -> {
  options.setEnablePerformanceV2(true);
  options.setEnableAppStartProfiling(true);
});
```

```kotlin {filename:MyApplication.kt}
import io.sentry.android.core.SentryAndroid

SentryAndroid.init(this) { options ->
  options.enablePerformanceV2 = true
  options.enableAppStartProfiling = true;
}
```

```xml {filename:AndroidManifest.xml}
<manifest>
    <application>
        <meta-data android:name="io.sentry.performance-v2.enable" android:value="true" />
        <meta-data android:name="io.sentry.traces.profiling.enable-app-start" android:value="true" />
    </application>
</manifest>
```

**对于iOS：**

- `>=7.2.0`用于[应用程序启动span](/platforms/apple/tracing/instrumentation/automatic-instrumentation/#app-start-tracing)的自动检测
- `>=8.21.0`用于应用程序启动分析

**对于React Native：**

- `>=5.24.0`用于[应用程序启动span](/platforms/react-native/tracing/instrumentation/automatic-instrumentation/#app-start-instrumentation)的自动检测

默认情况下，**应用程序启动**页面显示您所选时间范围内屏幕计数最高的两个版本的指标。要选择不同的版本集进行比较，请使用页面顶部的"版本选择器"。要更改应用程序启动类型（冷启动或热启动），请使用页面顶部的"应用程序启动"选择器。

图表显示以下指标（以冷启动为例）：

- 平均冷启动
    - 应用程序启动所需的总时间，按版本比较。
- 冷启动设备分布
    - 按[设备类别](/concepts/search/searchable-properties/#device-classification)（高、中、低或未知）分组的平均冷启动时间。

**您可能看不到任何数据的原因：**

- 您没有任何带有op `ui.load`的事务
- 您的SDK不满足最低SDK要求

底部的表格显示从一个版本到下一个版本发生变化的span，并允许您筛选特定的span操作和设备类别。能够缩小到特定事件样本有助于调试缓慢启动。

点击此表格右上角的"按事件"切换将显示按版本拆分的事件，您将能够看到所选两个版本之间启动时间的整体变化。

## Span详细视图

![Span详细视图示例](./img/app-starts-span-detail-view.png)

点击span描述会打开一个侧面板，显示span详细信息，包括您正在查看的每个版本中该span的平均持续时间和计数。

在下面的表格中，您将看到采样事件和配置文件（如果存在）的列表。通过比较两者，您将能够看到您正在查看的span的持续时间与所选两个版本中平均span的偏差程度。

Sentry会自动识别样本事件以帮助您调查性能问题。（它们显示为"平均持续时间"图表中的三角形。）为了给您提供准确的画面，在整个所选时间段内采样了比平均值快、比平均值慢和平均span持续时间的一系列样本。

您可以使用样本列表深入查看并比较给定屏幕内感兴趣的快速、平均和缓慢事件。

点击样本事件将带您进入[跟踪视图页面](/concepts/key-terms/tracing/trace-view/#产品演练-跟踪视图页面)的span瀑布中的查询span详细信息，或配置文件图标以查看事件的[火焰图](/product/explore/profiling/flame-charts-graphs/)。要获取不同的事件样本，请点击"尝试不同样本"按钮。
