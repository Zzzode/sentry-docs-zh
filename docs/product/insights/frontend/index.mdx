Sentry的 [**前端性能**](https://sentry.io/orgredirect/organizations/:orgslug/insights/frontend/) 页面为您提供了应用程序健康状况的概览。您将能够看到 **最佳页面机会**（最能帮助提高性能得分的改进）、**最耗时的资源**、**p50** 和 **p75持续时间** 等内容。

![前端性能概览页面。](../img/frontend-performance-overview.png)

您还可以深入了解Web Vitals、网络请求和资源，获取有关影响应用程序健康状况的潜在问题的详细信息。除了拥有专门的前端性能监控空间外，您还可以查看Sentry的 **Insights** 选项卡来监控后端、移动和AI性能。
